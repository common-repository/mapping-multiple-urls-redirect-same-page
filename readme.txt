== Description ==	
This Plugin used to make dynamic urls redirected to same page or post by adding some regular expression with page,post,category,tags,cutom post types,slug.

Note : If you using previous version then you have to upgrade by reistalling this plugin because its effect database also.

Major features in Mapping multiple URLs redirect same page include:

*   This Plugin used to make a alias link of your page,post,category,tag,custom post types.

*   This Plugin used to make dynamic urls redirected to same page or post by adding some regular expression with page,post,category,tags,cutom post types,slug.
